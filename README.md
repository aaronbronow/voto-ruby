# VotoRuby

A module for using the VOTO Mobile REST API.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'voto_ruby'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install voto_ruby
